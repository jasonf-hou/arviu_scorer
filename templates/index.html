Localization Error Scorer

Measure the distance between each participant's drawn mark and the corresponding reference target on ultrasound phantom drawings.

How it works:

  1. Each image shows a participant's drawing overlaid on the reference template.
  2. For each target, click two points: first the participant's drawn mark, then the actual reference target. A line will appear showing the error.
  3. Repeat for every visible target in the image.
  4. Click Submit & Next when done.
  5. Your progress is saved — you can close and resume later.
